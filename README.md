"# IhsanSouq" 

To run the API 
1.Navigate to API folder by:
~cd API
2.Restore dependencies:
~dotnet restore
3.Apply database migrations:
dotnet ef database update
4.Run the API:
dotnet run

To run the Angular Client:
1.Navigate to the client project folder:
~cd client
2.Install dependencies:
~npm install
3.Run server:
ng serve
